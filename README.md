# Portafolio
Este portafolio soy yo en una pagina web, donde expresa mi gusto por la naturaleza, anime y otras cosas. :3

## Tecnologías Usadas

- React + Vite
- JavaScript / JSX
- CSS3 personalizado
- Canvas API para animaciones
- Iconos SVG y assets visuales 
- Diseño responsivo y estético

## Ejecución
- cd portafolio   
- npm install
- npm run dev
